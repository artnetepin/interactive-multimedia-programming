Answers for the assignment:

  • HTML stands for Hyper-Text mark-up language and serves for building the structure of webpages and adding their content.
  • CSS stands for cascading stylesheet and serves for styling webpages.
  • JavaScript is a programming language for making the webpage dynamic and interactive.
  • HTTP stands for hypertext transfer protocol which serves for data transfer on the Web.
  • HTML5 is currently the latest version of HTML standard which is used in almost all the modern websites.
  • Dynamic HTML (DHTML) is a combination of technologies such as of HTML, CSS and JavaScript used to create dynamic and interactive webpages.

Coding conventions are sets of rules for writing the code used to improve code readability and make it maintainable.
