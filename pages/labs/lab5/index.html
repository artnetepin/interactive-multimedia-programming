Getting familiar with jQuery

What you can do with the jQuery?
jQuery is created to make working with DOM element simplier and is mostly used for creating animations, event handling and developing AJAX apps.
How it can help you as web developer?
It makes using JS for some purposes easier, faster and more comfortable especially when you're creating websites.
If you think our earlier participant list app, in what kind of issues you could use jQuery?
We could use jQuery selectors for getting elements (input, button), adding new elements from text input and jQuery methods for event handling.
What are CDNs? What kind of benefits you get if you use them?
CDNs are content delivery distributed networks used for Web content delivery with high performance and availability. The most common use of CDN for Web development is including different CSS/JS framework files (jQuery, Bootstrap, JS plugins) from the CDN. This increases, for instance, the perfomance of webpage in the way that it doesn't load those framework files from the local storage.
